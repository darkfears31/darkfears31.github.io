%x In C, %x is a format specifier used in the printf function to print an unsigned integer in hexadecimal format using lowercase letters.

We need to use this. So let’s connect to NetCat and Enter this into API key. We need to Enter lots of %x to provide almost all of the unsigned integers to find the flag. Just know that Flag will be in Hex.
